About Ocean Eddies Web Viewer

This version of the Ocean Eddies Web Viewer is not quite fully complete.

At the moment, there is currently no user-level documentation available for this web viewer.

Caveats with this version of the viewer:

  • The viewer needs to download all 39 MB of tracks data during initial startup.
  • In order to play the eddies data as an animation, you'll need a network connection that 1) has enough bandwidth to download the SSH data dynamically and 2) has a relatively low latency access link to this site. Otherwise, you'll just have to contend with stepping through the data one frame at a time.
  • There are a few missing features that would be nice to have.

Direct keyboard and mouse commands (those not accessed from GUI controls):

  • Click and drag on the map to pan.
  • Mouse wheel to zoom.

The tab, shift+tab, space bar, and enter keys can be used to access the GUI controls from the keyboard.
