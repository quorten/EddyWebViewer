About Ocean Eddies Web Viewer

This is an ALPHA version of the Ocean Eddies Web Viewer!

At the moment, there is currently no user-level documentation available for this web viewer.

You'll need a lot of RAM to run this viewer, something like 1 GB of spare RAM. You'll also need to make sure that you have a fast enough network connection to download 80 MB of data in less than 30 seconds, since the viewer needs to download all of the tracks data ahead of time. The viewer is also not as fast at rendering as it could be. These issues cannot be fixed until the next version.
